bin64ed is a small cross-platform utility that base64 encodes/decodes binary
files aswell as standard text.

Usage is simple: add files to the list or text in the box, select Encode or
Decode and off you go! bin64ed can handle large files just as well as small.

You can report bugs and make suggestions on the sourceforge project page at
https://sourceforge.net/p/base64-binary/tickets/
